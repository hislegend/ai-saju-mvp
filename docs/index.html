GitHub Pages Preview

AI 사주 MVP (MBTI 톤 개인화)

이 페이지는 공유용 프리뷰입니다. 현재 프로젝트는 Next.js API + Prisma DB 기반이라 GitHub Pages(정적 호스팅)에서 풀 기능 실행은 불가능합니다.

README 보기 GitHub 저장소

구현된 IA

  • / 메인 랜딩
  • /quick 무료 1분 사주
  • /premium/:slug 유료 랜딩
  • /checkout/:readingId 결제/리딤
  • /result/:readingId 결과
  • /mypage 재조회/내역

핵심 차별점

  • MBTI 직접 입력 또는 4문항 간이 진단
  • 동일 사주 데이터에서 MBTI별 톤/행동 가이드 분기
  • 무료 진입 → 유료 업셀 퍼널 이벤트 추적

풀 기능 공유 링크가 필요하면

GitHub Pages 대신 서버 실행이 가능한 배포(Vercel/Render/Railway)로 배포하면 결제/리딩/API까지 그대로 공유할 수 있습니다. 원하면 제가 바로 배포용 설정까지 이어서 진행하겠습니다.
